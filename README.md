# 🚀 Машина Времени: Викторина по Истории России

Интерактивное учебное пособие по предмету «Окружающий мир» для учеников 4 класса.

## 📝 Описание проекта
Этот проект представляет собой одностраничное веб-приложение (SPA), созданное для закрепления знаний по ключевым историческим темам программы начальной школы. Игра построена на механике викторины с мгновенной обратной связью и системой поощрений.

### 📚 Охваченные темы:
1.  **Отечественная война 1812 года** — Бородинское сражение и полководцы.
2.  **Страницы истории XIX века** — отмена крепостного права.
3.  **Россия вступает в XX век** — последний император.
4.  **Страницы истории 1920-1930-х годов** — великие стройки (индустриализация).
5.  **Великая Отечественная война** — память и победа.
6.  **Страна, открывшая путь в космос** — первый полет человека.

## 🎮 Игровые механики
- **Сбор наград:** за каждый правильный ответ ребенок получает золотую монету (💰).
- **Мгновенная проверка:** система сообщает об ошибке сразу, позволяя попробовать еще раз.
- **Финал:** виртуальный кубок и звание «Знатока Истории» при успешном прохождении.

## 🛠 Технологии
Проект выполнен с использованием чистого стека технологий без внешних зависимостей:
- **HTML5** — структура и контент.
- **CSS3** — современный дизайн, адаптивная верстка и анимация кнопок.
- **JavaScript (Vanilla JS)** — логика переключения вопросов и система подсчета очков.

## 🚀 Как запустить
1. Скачайте файл `index.html`.
2. Откройте его в любом современном браузере (Chrome, Firefox, Safari, Edge).
3. Не требует установки сервера или подключения к интернету.

## 💡 Цель проекта
Сделать процесс обучения легким и увлекательным, превращая проверку знаний из стрессового теста в небольшое приключение.
